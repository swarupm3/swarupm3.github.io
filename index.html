GPU Acceleration for STAPL Parasol Laboratory, UIUC

GPU Interoperability for Standard Template Adaptive Parallel Library (STAPL) — Parasol Lab, UIUC

This project explores how GPUs can be integrated into the Standard Template Adaptive Parallel Library (STAPL) to speed up for_each, an algorithm in C++ STL. By using CUDA and benchmarking on an 8-GPU NVIDIA H100 system, we found that using pinned and zero-copy memory dramatically reduces memory copy overhead and delivers 10-100x speedups over CPU execution for complex workloads. Overall, we observed that GPU acceleration can effectively improve runtime of common STL algorithms. I am currently working on accelerating other STL-equivalent STAPL algorithms such as reverse.
